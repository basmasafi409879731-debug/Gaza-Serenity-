#  Gaza Serenity – دليل الأماكن الهادئة في غزة

مشروع Gaza Serenity هو موقع ويب بسيط يعرض مجموعة من الأماكن الهادئة في غزة، مع وصف وصور وفيديو وجدول لأفضل أوقات الزيارة.  
تم بناء الموقع باستخدام **HTML + CSS + JavaScript فقط** بدون أي مكتبات خارجية.

##  الفئة المستهدفة
المشروع موجّه للمستخدمين الذين يبحثون عن أماكن هادئة في غزة،  
وللمهتمين بتجربة تصفح بسيطة ومريحة تعتمد على تصميم نظيف وواضح.  
كما يناسب الطلاب والمطورين الذين يرغبون في رؤية مثال عملي على بناء موقع كامل بدون مكتبات خارجية.

##  المميزات
- تصميم هادئ يعكس هوية المشروع.
- كروت تُنشأ ديناميكيًا عبر JavaScript.
- تأثير ظهور تدريجي (Fade-in).
- وضع ليلي (Dark Mode).
- تصميم متجاوب مع الشاشات المختلفة.
- فيديو وصور عالية الجودة.
- نموذج تواصل بسيط.

##  التقنيات المستخدمة
- **HTML5** لبناء الهيكل.
- **CSS3** لتصميم كامل بدون Bootstrap.
- **JavaScript** للتحكم في المحتوى والأنيميشن والوضع الليلي.

##  الهدف
تقديم تجربة هادئة وبسيطة للمستخدم، وإظهار جمال غزة من خلال أماكنها الهادئة،  
مع تطبيق مبادئ تصميم نظيف باستخدام أساسيات الويب فقط.
