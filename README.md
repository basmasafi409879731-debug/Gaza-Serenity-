#  غزة الهادئة | Gaza Serenity

##  وصف المشروع
مشروع "غزة الهادئة" هو موقع بسيط وجذاب يهدف إلى عرض مجموعة من الأماكن الهادئة في قطاع غزة، مع توفير معلومات عن أفضل الأوقات لزيارتها، بالإضافة إلى فيديو هادئ ونموذج تواصل.  
الموقع مصمم ليكون مريح بصريًا، سهل الاستخدام، ويقدم تجربة هادئة للمستخدم.

---

##  مكوّنات المشروع
يتكون المشروع من الأقسام التالية:

### ✔ الهيدر (Header)
- شعار الموقع  
- قائمة تنقل داخلية  
- زر تغيير النمط (الوضع الداكن)

### ✔ قسم الهيرو (Hero Section)
- صورة خلفية كبيرة  
- عنوان رئيسي  
- زر ينقل المستخدم لقسم الأماكن

### ✔ قسم الأماكن (Spots Section)
- 4 بطاقات تحتوي على:
  - صورة  
  - اسم المكان  
  - وصف بسيط  

### ✔ قسم الفيديو (Video Section)
- فيديو هادئ من غزة داخل صندوق منسق

### ✔ قسم الجدول (Table Section)
- جدول يعرض:
  - اسم المكان  
  - مستوى الهدوء  
  - أفضل وقت للزيارة  

### ✔ قسم التواصل (Contact Section)
- نموذج يحتوي على:
  - الاسم  
  - البريد الإلكتروني  
  - الرسالة  

### ✔ الفوتر (Footer)
- نص بسيط في أسفل الصفحة

---

##  التقنيات المستخدمة

### ✔ HTML5  
لبناء الهيكل الأساسي للموقع.

### ✔ CSS3  
لتنسيق الألوان والخلفيات والبطاقات والجدول والفورم.

### ✔ Bootstrap  
لتسهيل تقسيم الصفحة باستخدام Grid System.

### ✔ JavaScript + jQuery  
لتفعيل ميزة **الوضع الداكن (Dark Mode)**.

---

##  ميزة الوضع الداكن (Dark Mode)
تم استخدام jQuery لإضافة وحذف كلاس `dark-mode` من عنصر `<body>` عند الضغط على زر "تغيير النمط".

الكود المستخدم:

```javascript
$("#modeBtn").click(function () {
    $("body").toggleClass("dark-mode");
});
